# Metropolitan League — Website

Static multi-page site (no build step).

## Pages
- `index.html` — Home
- `about.html` — About the League
- `rules.html` — Format & Rules

## Deploy to GitHub Pages
1. Create a repo and upload the **contents** of this folder (index.html must be at the repo root).
2. Settings → Pages → Deploy from a branch → `main` / `(root)` → Save.
3. Live at `https://<username>.github.io/<repo>/`

Images are embedded; fonts load from Google Fonts. `.nojekyll` keeps GitHub Pages from altering files.
